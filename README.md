# Generic Course — Institutional Landing Page

A generic institutional landing page for an online course platform, built with vanilla HTML, CSS and JavaScript.

## 🔗 Live Demo

[View project](https://guilhermeseckler.github.io/landing-page-institucional/)

## 📋 About

This project was developed as part of my ADS (Systems Analysis and Development) coursework. The goal was to build a complete, responsive institutional landing page using only vanilla technologies — no frameworks or libraries.

## ✨ Features

- Responsive layout with media queries and `clamp()` for fluid typography
- Sticky header
- FAQ accordion — opens and closes answers on click, one at a time
- Registration modal using the native `<dialog>` API
- BEM methodology for CSS class naming
- CSS custom properties for consistent design tokens

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, custom properties, clamp, media queries)
- JavaScript (DOM manipulation, classList, native dialog API)
